## Synopsis

Personal Maven Archetype for OSS Projects.

## Code Example

``` shell
mvn -B archetype:generate\
    -DarchetypeGroupId=net.middell\
    -DarchetypeArtifactId=oss-archetype\
    -DarchetypeVersion=1.0-SNAPSHOT\
    -DgroupId=...\
    -DartifactId=...
```

## Installation

``` shell
mvn install
```
